# 选择结构与循环结构

## 1. 选择结构程序设计
  - `if` 与 `else`
    - `else` 不能单独出现；
      ``` C
        if(i < 0){
          // ...语句1
        }
        else{
          // ...语句2
        }
      ```
    - `if` 与 `else if`；
      ``` C
        if(i <= 0){
          // ...语句1
        }
        else if(i > 0 && i <= 10){
          // ...语句2
        }
        else if(i > 10 && i <= 15){
          // ...语句3
        }
        else{
          // ...语句4
        }
      ```
    - `if` 内嵌套 `if` 和 `else`。
      ``` C
        if(i < 0){
          if (j > 3){
            // ...语句1
          }
          else{
            // ...语句2
          }
        }
      ```

## 2. 循环结构程序设计
### 2.1 `while` 循环
  - 出现死循环的原因：
    1. `while()` 后面直接加 `;`
        ``` C
          // 错误示范，会出现死循环的现象
          while(i < 10);
        ```
    2. `while` 语句体内没有让表达式趋近于假的操作。
  - 若执行后出现终端卡住的情况，原因如下：
    - 出现了死循环，此时无法输入任何内容；
    - 卡在 `scanf`，这种情况下，手动输入内容的话，可输入进去的。

### 2.2 `for` 循环
  - 一般形式为：
    ``` C
      for(表达式1 ; 表达式2 ; 表达式3) 
        语句;
      // ...其他语句
    ```
  - 执行过程：
    1. 求解表达式1；
    2. 求解表达式2，若其值为真，则执行 `for` 语句中指定的内嵌语句，再执行步骤 3；若其值为假，则结束循环，并转到步骤 4；
    3. 求解表达式3，并转回步骤 2 继续执行；
    4. 循环结束，执行 `for` 语句下面的其他语句。
